git bash
① cd +フォルダのフルパス（フォルダに移動する）
② git status (変更されたファイルを確認する）
③ git add +ファイル名 (変更内容をインデックスに追加する）
④ git commit (ファイルの変更予約を全て実行する）
⑤ git push origin +ブランチ名 (リモートサーバに対してアップロードする,从将本地的分支版本上传到远程并合并）
